# tmux

## Key bindings

Prefix: `Ctrl+Space` (not default Ctrl+b)

| Key | Action |
|-----|--------|
| `prefix + r` | Reload config |
| `prefix + h/j/k/l` | Navigate panes (vim-style) |
| `prefix + "` | Split horizontal (current dir) |
| `prefix + %` | Split vertical (current dir) |

Vi mode in copy-mode: `v` to select, `y` to yank.

## Plugins (TPM)

- vim-tmux-navigator
- tmux-yank
- catppuccin theme

First run: `prefix + I` to install plugins.
