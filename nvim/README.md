# Turner's Neovim Config


## Navigation

#### Open File Explorer Left
`<C-n>`
#### "Hop" in Files
`<Leader>h`

## Debugging

#### Toggle Breakpoint
`<Leader>dt`
#### Continue
`<Leader>dc`
#### Terminate
`<Leader>dx`
#### Step Over
`<Leader>do`

## Formatting
#### Format the Current File
`<leader>gf`
